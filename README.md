# 🎮 Simon Game

An interactive and fun memory-based game built using **HTML, CSS, and JavaScript**. The goal of the game is to repeat the sequence of colors shown by the system. As the game progresses, the sequence becomes longer and more challenging.

---

## 🚀 Features

* 🎯 Simple and engaging gameplay
* 🎨 Interactive UI with animations and sound effects
* 📈 Increasing difficulty levels
* ⚡ Real-time user interaction using JavaScript
* 💻 Fully responsive design

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the game
* **CSS3** – Styling and animations
* **JavaScript (ES6)** – Game logic and interactivity

---

## 🎮 How to Play

1. Press any key (or click Start button) to begin the game
2. Watch the pattern of colors displayed
3. Repeat the same sequence by clicking the buttons
4. Each correct sequence increases the level
5. If you press the wrong button, the game ends
